Weather Telegram Bot

Simple Telegram bot for getting weather information.

Stack:

Python
aiogram 3
requests
python-dotenv

Features:

current weather
async handlers
environment variables support
logging

Installation:

git clone https://github.com/sladkozubik/shitty.git
cd shitty
pip install -r requirements.txt

Configuration:

Create .env file:
BOT_TOKEN=your_token

Run:
python main.py
